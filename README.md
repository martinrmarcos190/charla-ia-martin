# Charla IA para devs

Materiales de las charlas y el workshop de IA para desarrolladores: decks (HTML/reveal), notas del orador y recursos.

## Contenido

| Carpeta | Descripción |
|---------|-------------|
| **Fundamentos de la IA/** | Charla introductoria. Deck en `index.html`, spec y notas del orador. |
| **Estado Actual, Herramientas y Skills/** | Charla sobre el estado actual del ecosistema, herramientas y skills. Deck en `index.html`. |
| **Workshop Práctico SDD + Skills/** | Workshop práctico de Spec-Driven Development + Skills. Slides en `slides/`, consignas en `recursos/`. |
| **vendor/** | Librerías de terceros (reveal.js, etc.) usadas por los decks. |
| **OLD/** | Versiones anteriores del material. |
| `Charla_IA_V3.pdf` | Versión PDF de la charla. |

## Cómo ver los decks

Cada deck es un `index.html` autocontenido. Abrir directamente en el navegador, o servir la carpeta localmente:

```bash
python3 -m http.server 8000
# luego abrir http://localhost:8000
```

## Notas

- Los decks usan formato cuadrado, pensado para presentar en Google Meet.
- Las notas del orador están en los archivos `notas_orador*.html` / `*_notas_orador.md` de cada charla.
